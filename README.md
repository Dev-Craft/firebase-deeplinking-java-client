# Java Client for Firebase Dynamic Links

## Gradle
```
repositories {
    mavenCentral()
}

dependencies {
    compile ('io.github.dev-craft:firebase.deeplinking.client:1.0')
}
```

## Maven
```
<dependency>
  <groupId>io.github.dev-craft</groupId>
  <artifactId>firebase.deeplinking.client</artifactId>
  <version>1.0</version>
</dependen
```
